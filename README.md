# Nasa-photo-of-the-day
First-ever WEBSITE of mine, built without any AI. Space-themed website that fetches NASA's photo of the day using an API. Displays space imagery of planets with scientific explanations.

main features
API integration
interactive date navigation
automatic media adaptation
good error handling

Used:
HTML5
CSS3
JavaScript
NASA APOD API

to run the project make sure you have vs code

1. clone the repository to your own machine
terminal:
bash
git clone https://github.com/Tr33Fr0g/Nasa-photo-of-the-day.git
